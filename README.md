## twinNNPPI
twinNNPPI is a deep learning-based framework designed for the prediction of protein-protein interactions (PPIs). By combining traditional sequence-based numerical features with state-of-the-art protein language model embeddings, the tool provides a robust approach to identifying functional and structural associations between proteins.

## Core Features
Hybrid Feature Extraction: Integrates multiple representation strategies to capture diverse protein characteristics:
Hand-crafted Sequence Features: Utilizes Auto-Correlation (AC), Auto-Cross Covariance (ACC), and Conjoint Triad descriptors.
Raw Language Model Embeddings: Leverages per-protein embeddings generated by the ProtT5 model for deep semantic representation.
Unified Prediction Pipeline: Processes protein pairs through a specialized neural network architecture to determine interaction probability.
Scalable Data Source: Built to utilize sequence information directly from UniProt.

## Methodology
The twinNNPPI architecture processes a pair of proteins using two parallel information streams:
Sequence-Based Stream: Extracts evolutionary and physicochemical information using AC, ACC, and Conjoint Triad methods.
Embedding-Based Stream: Captures high-level contextual information from the ProtT5 transformer model, which is pre-trained on billions of protein sequences.
Classification: The features from both proteins are concatenated or fused, then passed through dense neural layers to output an interaction score.

## Getting Started
### Prerequisites
Python 3.8+

PyTorch

Transformers (for ProtT5 access)

Scikit-learn

NumPy/Pandas

## Usage
Input protein sequences in FASTA format or provide UniProt IDs to fetch sequences and generate predictions.
